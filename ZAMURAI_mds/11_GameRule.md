# 🎮 「ちんちん侍 VS トゥントゥンサフール」（仮）

**コンセプト:**

「ちんちん侍」×「Lethal Company的カオス協力」×「青鬼的ホラー」

声とリズムで儀式を完遂し、迫りくる怪異「トゥントゥンサフール」を退けろ。

---

## 🏛️ マップ構成・オブジェクト

マップは「広大な館」。複雑すぎず、シンプルに

- **儀式部屋（中央）:** 椅子が4つ。ここでの「ちんちん侍」が勝利条件。
    
- **ブレーカー（3箇所）:** * 定期的にダウンし、マップが真っ暗になる。
    
    - 停電中は儀式のポイントが入らないため、誰かが直しに行く必要がある。
        
- **閉じ込め部屋（1箇所）:** * 敵を誘い込んでロックできるトラップ部屋。
    
    - **耐久値:** ランダムな時間で破壊され、敵が再出現する（扉を叩く予兆あり）。
        

---

## 👺 敵：トゥントゥンサフール

- **行動:** 音に反応して接近。儀式中のプレイヤーに「少しの音（囁き、ノック）」で揺さぶりをかける。
    
- **殺意:** 儀式が進む（ポイントが溜まる）ほど、また「おちんちん」等の大声に反応して凶暴化。
    
- **リスポーン:** 捕まったプレイヤーは一定時間後に復活。3人でも儀式は可能だが、効率が大幅に落ちる。

## 🎤 儀式システム（ちんちん侍）

マイク入力による「声操作」が基本。指差し（エイム）で次の人を指定する。

### 1. 基本コマンドとスコア

コマンド「ちんちん」「おちんちん」「侍」「ちんちん侍」
リアクション「びろーん」「シャキーン」「ちんちん侍」

うまく反応できないとミス、儀式は最初から
・相手にコマンドを渡せなかった
・リアクションを返せなかった

| **アクション** | **発声内容** | **スコア** | **効果・リスク**           |
| --------- | -------- | ------- | -------------------- |
| **ちんちん**  | 「ちんちん」   | 1点      | 基本。リスクなし。            |
| **おちんちん** | 「びろーん」   | 5点      |  敵を呼び寄せる。            |
| **侍**     | 「シャキーン」  | 5点      | 敵を呼び寄せる。             |
| **ちんちん侍** | 「ちんちん侍」  | 10点     | **連帯責任。** 全員同時入力で爆風。 |

### 2. 特殊ルール

- **おすすめワード:** 画面上部にランダム表示される。その通りに発声すると**スコア2倍**。
    
- **ちんちん侍チャンス（1/10）:** * 突如発生するコンボチャンス。炸裂すれば**3倍**。
    
    - 連続発生時は**4倍、5倍**と倍率が跳ね上がる。
        
- **リズム加速:** 儀式が進むにつれてテンポが高速化。
    
    - 一定時間反応がないと「ミス（点数停滞）」。
        
    - 初速とトップスピードには制限あり。

## 🚀 ゲームフロー

1. **探索:** ブレーカーを上げ、トラップを準備する。
    
2. **隔離:** 誰かがデコイ（おちんちん）になり、敵を閉じ込め部屋へ誘導・ロック。
    
3. **儀式:** 全員で中央に集まり、ちんちん侍を開始。
    
    - 「声操作」のため、戦略的な相談は物理的に困難（空気読みゲー）。
        
    - 停電や扉の破壊音にビビりながらリズムを維持する。
        
4. **クリア：ちんちん侍フィーバー:**
    
    - 儀式完了で無敵モード突入。
        
    - 「ちんちん侍」連打で衝撃波を出し、トゥントゥンサフールをボコボコにして脱出。
